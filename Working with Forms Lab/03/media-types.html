Textarea

The textarea element represents a multi-line plain-text editing control, useful when you want to allow users to enter a sizeable amount of free-form text, for example a comment on a review or feedback form.

Attribute - rows

The rows attribute specifies the number of visible text lines for the control.

<textarea rows="5" cols="45"></textarea>

Attribute - cols

The cols attribute specifies the visible width of a text area.

<textarea rows="5" cols="45"></textarea>

Attribute - placeholder

The placeholder attribute provides a hint to the user of what can be entered in the field.

<textarea rows="5" cols="45" placeholder="Enter your text here..."></textarea>
